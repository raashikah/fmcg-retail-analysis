# FMCG Retail Sales Analysis & Customer Segmentation

An end-to-end Machine Learning project analysing retail sales performance, customer segmentation, and sales forecasting using Python.

## Project Overview
- **EDA** — Sales trends, category performance, regional breakdown
- **Customer Segmentation** — K-Means clustering to identify High, Mid, and Low value segments
- **Sales Forecasting** — Random Forest Regressor with 99% accuracy

## Tech Stack
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (KMeans, RandomForest)

## Key Results
- Total Sales Analysed: Rs. 5.6 Crore
- Profit Margin: 17.1%
- ML Model Accuracy: 99% R2 Score
- Top Category: Beverages
- Top Region: West

## How to Run
pip install pandas numpy matplotlib seaborn scikit-learn
python retail_analysis.py
